# Ship it

Vibe coding 完成網站後
還有以下幾件事情可以做

1. 把程式碼上到github (保管你的程式碼)
2. 部署到網路上 (這樣網站才可以分享給你的朋友)
3. 使用你自己的域名 (可選 如果只是給朋友或同事用就不需要)

## Github

這是一個全世界工程師都在用的網站
可以幫你保管你的程式碼
建議你不管怎麼樣都去辦一個帳號

## 部署

Vibe coding開發的時候
通常你的網址會是 `localhost`
但是就這樣直接拿給你的朋友
他什麼都看不到

所以你需要找地方部署
但是如果你不知道框架 伺服器之類的名詞
我會建議你部署到Vercel
因為他可以相容最多的框架

## 域名

域名的目的是讓你有一個更好記的網址
部署在Vercel的話
你的網址看起來會是
https://vibeship-w1z3mwz7.vercel.app/

如果只是給認識的人用當然沒問題
但是如果你想要給其他人
那麼我就會建議你去買一個網址
這裡建議去Cloudflare買
這樣之後設定才會方便

## 因此我做了一個三分鐘讓你的Vibe coding上線的工具

https://sheepit.cc

步驟只有三個
1. 上傳你的資料夾
2. 幫你上傳到github
3. 幫你部署到vercel

如果你需要 他也可以幫你綁定你的網址
