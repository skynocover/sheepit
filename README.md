## ✨ 功能特色

- **環境變數自動處理** — 自動讀取專案中的 `.env` 檔，部署時同步到 Vercel
- **專案管理 Dashboard** — 查看所有專案狀態、部署記錄、即時追蹤部署進度
- **重新部署** — 上傳新版本的程式碼，一鍵更新已部署的網站
- **自訂域名** — 透過 Cloudflare 一鍵綁定自訂域名，自動設定 DNS
- **QR Code 分享** — 產生 QR Code，方便用手機掃碼或分享給朋友
- **Gallery 作品牆** — 將作品公開到 Gallery，展示給社群看

## 🤔 這些東西是什麼？

### GitHub

全世界工程師都在用的程式碼保管庫。
把程式碼放上去，就不怕電腦壞掉、檔案不見。
**建議不管怎樣都去辦一個帳號。**

### Vercel

幫你把網站放到網路上的服務。
Vibe coding 開發時，網址通常是 `localhost`——只有你看得到。
Vercel 會給你一個真正的網址，任何人都能打開。

### 自訂域名（可選）

部署後的網址會長這樣：

```
https://your-project-abc123.vercel.app
```

給朋友用完全沒問題。但如果你想要更好記的網址，可以去買一個域名。
**推薦在 [Cloudflare](https://www.cloudflare.com/) 購買**，之後設定最方便。

## ❓ 常見問題

### 需要先辦什麼帳號？

- **GitHub 帳號** — 必須，用來存放你的程式碼
- **Vercel 帳號** — 必須，用來部署網站（可以用 GitHub 帳號直接登入）
- **Cloudflare 帳號** — 可選，只有要綁定自訂域名時才需要

### 免費嗎？

- **Ship It** — 免費使用
- **GitHub** — 免費
- **Vercel** — 個人專案免費（有流量限制，但一般夠用）

### 我的程式碼安全嗎？

Ship It 只是幫你把檔案推到 **你自己的** GitHub 帳號，我們不會儲存你的程式碼。
整個流程你都可以在 GitHub 和 Vercel 上看到發生了什麼事。

### 部署後可以更新嗎？

可以！在 Dashboard 的專案頁面中，你可以上傳新版本的程式碼重新部署。
